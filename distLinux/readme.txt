Hello, this is DSYSender. This readme is made for Re Requiem but may be included with other versions of the mod. I'd check the nexus mods page for the specific
game you are looking to play for more details.

If you are on Windows OS please don't use this. It is not necessary, Lunatii's Mod works natively with DualsenseY-V2.


Installation:
Move the three files included into {gamefolder}/reframework/data/dualsenseX/
You should see a file called "Payload.json" in there from Lunatii's mod.
Make sure all 3 files are in there.


Launching The Game:
1. Launch DSY-V2
2. Launch the game or this, it doesn't matter. When you want to launch this run the file "dsySenderRunLinux.sh" as this script is built to be run in konsole. It won't work alone.
3. That's it!

Note: 
Make sure you got all the prerequisites listed installed and ready from Nexus or Github,
DualsenseY-V2
reframework
Lunatii's DualsenseX Adaptive Trigger Mod for whichever game you are playing.


There is a chance I port this to other games (red dead 2) or other games with AT mod support and I might forget to update this readme.txt

Just make sure you read the description and check the requirements for that game
